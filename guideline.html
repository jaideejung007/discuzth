Discuz!

ฟอรั่มดิสคัสไทย Facebook ดิสคัสไทย คำแนะนำในการติดตั้ง ดาวน์โหลด Discuz! X3.4 ภาษาไทย
การติดตั้งใหม่

  1. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดใน upload ไปยังเซิร์ฟเวอร์ของคุณ โดยใช้โปรแกรม FTP ตามสะดวกของคุณ เช่น โปรแกรม FileZilla เป็นต้น (ไม่จำเป็นต้องอัปโหลดโฟลเดอร์ readme และไฟล์ utility.html)
  2. เปิดเบราว์เซอร์เพื่อทำการติดตั้ง Discuz! X3.4 โดยไปที่ yourdomain.com/install เพื่อเริ่มต้นขั้นตอนการติดตั้ง แล้วทำตามหน้าจอที่ปรากฏ

รายละเอียดทั้งหมดดูที่นี่: การติดตั้ง Discuz! X3.4 แบบละเอียด


การอัปเกรด

อัปเกรดจาก X3.2, X3.3 เป็น X3.4

  1. สำรองฐานข้อมูล
  2. สร้างโฟลเดอร์ old ขึ้นมาใหม่ แล้วทำการย้ายไฟล์ทั้งหมดไปยังโฟลเดอร์ old ยกเว้นโฟลเดอร์ต่อไปนี้ data, config, uc_client, uc_server
  3. อัปโหลดไฟล์ X3.4 ทับตัวเก่าบนเซิร์ฟเวอร์
  4. เช็คสิทธิ์โฟลเดอร์ ./data/* และ ./uc_server/data/* และ ./uc_client/data/* ให้เป็น 0777 (รวมทั้งโฟลเดอร์ย่อยด้วย)
  5. ลบปลั๊กอินที่เกี่ยวข้องกับแพลตฟอร์มคลาวด์ของดิสคัสจีน (เนื่องจากยกเลิกให้บริการแล้ว) ดาวน์โหลดไฟล์ utility จากนั้นอัปโหลดไฟล์ utility/clearcloud.php ไปยังโฟลเดอร์ install
  6. รันไฟล์ yourdomain.com/install/clearcloud.php
  7. ลบโฟลเดอร์ ./install ออก
  8. เช็คสิทธิ์โฟลเดอร์ ./data/* และ ./uc_server/data/* และ ./uc_client/data/* ให้เป็น 0777 อีกครั้ง (รวมทั้งโฟลเดอร์ย่อยด้วย)
  9. เข้าระบบหลังบ้าน AdminCP ไปที่เมนู เครื่องมือ -> อัปเดตแคช -> ติ๊กเลือกทั้งหมด -> กดตกลง
  10. ตรวจสอบปลั๊กอินและเทมเพลท (สกิน) เดิมว่ารองรับ X3.4 หรือไม่ หากรองรับให้ทำการย้ายจากโฟลเดอร์ปลั๊กอิน old/source/plugin/* และจากโฟลเดอร์เทมเพลท (สกิน) old/template/* ไปยังตำแหน่งเดิมได้ (คือ source/plugin/* และ template/*)

อัปเกรดจาก X3, X3.1 เป็น X3.4

  1. สำรองฐานข้อมูล (หากเป็นไปได้ ให้สำรองไฟล์ทั้งหมดในเว็บด้วยก็ดี)
  2. สร้างโฟลเดอร์ old ขึ้นมาใหม่ แล้วทำการย้ายไฟล์ทั้งหมดไปยังโฟลเดอร์ old ยกเว้นโฟลเดอร์ต่อไปนี้ data, config, uc_client, uc_server
  3. เริ่มอัปโหลดไฟล์ Discuz! X3.4 ภาษาไทยล่าสุด ทับไฟล์ตัวเก่าในเซิร์ฟเวอร์
  4. เช็คสิทธิ์โฟลเดอร์ ./data/* และ ./uc_server/data/* และ ./uc_client/data/* ให้เป็น 0777 (รวมทั้งโฟลเดอร์ย่อยด้วย)
  5. เริ่มต้นการอัปเกรดเป็น Discuz! X3.4 ให้ดาวน์โหลดไฟล์ utility จากนั้นอัปโหลดไฟล์ utility/update.php และ utility/clearcloud.php ไปยังโฟลเดอร์ install
  6. รันไฟล์ yourdomain.com/install/update.php
  7. ลบปลั๊กอินที่เกี่ยวข้องกับแพลตฟอร์มคลาวด์ของดิสคัสจีน (เนื่องจากยกเลิกให้บริการแล้ว) โดยรันไฟล์ yourdomain.com/install/clearcloud.php
  8. ลบโฟลเดอร์ ./install ออก
  9. เช็คสิทธิ์โฟลเดอร์ ./data/* และ ./uc_server/data/* และ ./uc_client/data/* ให้เป็น 0777 อีกครั้ง (รวมทั้งโฟลเดอร์ย่อยด้วย)
  10. เข้าระบบหลังบ้าน AdminCP ไปที่เมนู เครื่องมือ -> อัปเดตแคช -> ติ๊กเลือกทั้งหมด -> กดตกลง
  11. ตรวจสอบปลั๊กอินและเทมเพลท (สกิน) เดิมว่ารองรับ X3.4 หรือไม่ หากรองรับให้ทำการย้ายจากโฟลเดอร์ปลั๊กอิน old/source/plugin/* และจากโฟลเดอร์เทมเพลท (สกิน) old/template/* ไปยังตำแหน่งเดิมได้ (คือ source/plugin/* และ template/*)
  12. กลับไปที่หน้าหลัก แล้วกดปุ่ม Ctrl + F5 จะเป็นการเคลียร์ข้อมูลเก่าจากเบราว์เซอร์ เพื่อเรียกไฟล์ภาพและอื่น ๆ จากเว็บไซต์เราใหม่

รายละเอียดทั้งหมดดูที่นี่: การอัปเกรดจาก Discuz! X3.x (ได้แก่ X3, X3.1) ไปยัง Discuz! X3.4

อัปเดตไฟล์แพทช์ (สำหรับผู้ที่ใช้งาน Discuz! X3.4 อยู่แล้ว)

  1. ดาวน์โหลดไฟล์แพทช์จากกระทู้ Discuz! X3.4 ภาษาไทย
  2. แยกไฟล์ ZIP แล้วอัปโหลดทับไฟล์เก่าบนเซิร์ฟเวอร์ของคุณ (แนะนำให้สำรองข้อมูลเว็บไซต์ก่อนดำเนินการ)
  3. เข้าสู่ระบบ AdminCP แล้วอัปเดตไฟล์แคช
